# import-tickers

Download information about tradeable assets from multiple sources

Sources include:

1. Polygon.io
2. Tiingo
3. OpenFIGI
4. Yahoo Finance (https://query2.finance.yahoo.com)

Information Retrieved:

* Ticker
* Asset Type (Common Stock, Mutual Fund)
* Asset Name
* Primary Exchange
* Logo URL
* Corporate URL
* Listing Date
* Delisting Date
* Composite FIGI
* Share Class FIGI
* CUSIP
* ISIN
* CIK
* Sector
* Industry
* Headquarters Location
* Similar Tickers
